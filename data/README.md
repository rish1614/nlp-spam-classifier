## 📂 Directory Structure

We strictly follow the **Immutable Raw Data** principle. 

```text
data/
├── raw/                 # Committed to GitHub
│   ├── spam.tsv         # The original, untouched dataset from UCI
│
├── processed/           # Ignored by GitHub (.gitignore)
│   ├── cleaned_spam.csv # Generated by our Python pipeline
│
└── README.md            # This documentation file
